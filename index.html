使用说明与用户协议

版本：v0.2.22 | 更新日期：2026-04-06 | GitHub 仓库

欢迎来到齐喜的网站，请先阅读以下使用说明、用户协议与隐私条款后使用本网站。

我们使用 HTML 标签来表明重点内容。如果您在阅读时发现了形如 <strong> 的标记，请将其理解为视觉强调。


一、开源协议

本网站源代码采用 MIT 开源协议发布。 您可以自由查看、使用、修改和分发源代码，但需保留原始版权声明。 查看完整协议文本。

本站托管于 GitHub，由 Netlify 部署，认准网址 nb-qx.top。（近期发现对应.com网站是不良网站）


二、功能与服务

2.1 产品中心

产品中心 展示虚拟化学产品，提供： 图片、体验链接、下载链接 和 视频链接。

体验链接可试用虚拟产品一分钟（非手动设置）。如需长时间使用，请选择 下载 或使用微信小程序。有关 NOBOOK 的使用与分享，请参阅 产品帮助手册。

下载将跳转至 123 网盘。获取 .nbc 文件后，需在 网页版 NOBOOK 化学虚拟实验 中手动导入。

2.2 兑换码功能

兑换码功能持续研发中。我们使用 SHA-256 算法验证兑换码有效性。 此功能不具有商业、赌博等性质，仅供娱乐。

2.3 天气服务

天气信息来源于 Open-Meteo 开源项目。所设目标位置为随机生成，不指代任何真实个人住址。

2.4 评论与互动

本网站提供以下评论方式：

  • Giscus 评论：基于 GitHub Discussions，需遵守 GitHub 服务条款。 评论数据存储于 GitHub 平台。
  • 表单评论：通过 Netlify Forms 提交，需经人工审核后展示。 请勿提交敏感个人信息。

您承诺评论内容不包含：违法信息、人身攻击、广告 spam、冒充他人。 IP 地址将被记录用于反 spam，不对第三方公开。


三、重要安全警告

⚠️ 虚拟产品仅在 NOBOOK 平台内运行

所有"洗钉机"、"洗钉液"、"镀铜水"等产品均为虚拟化学实验。
严禁在现实中复刻、制作、购买原料尝试！ 化学实验需在专业指导下于安全场所进行，否则可能造成严重人身危险！


四、隐私条款与数据收集

4.1 收集的信息类型

信息类型 用途 存储位置
设备唯一标识符 访问统计、防刷 Vercount / Cloudflare
粗略位置（国家/地区） 天气服务、区域分析 Open-Meteo / 本地缓存
系统与浏览器版本 兼容性优化 Cloudflare Insights
访问时间与状态 网站改进分析 Netlify / Cloudflare
评论内容（如提交） 展示与互动 GitHub Discussions / Netlify

注：v2.0.0 版本将引入数据库存储，届时本条款将更新以反映变化。

4.2 数据使用原则

  • 收集的数据仅用于分析统计、防刷、网站改进
  • 不会进行买卖、交易或向第三方泄露
  • 符合《中华人民共和国个人信息保护法》等相关法律规定

4.3 儿童隐私保护

本网站面向全世界各类人群（爬虫除外）。 不会主动收集 14 岁以下儿童的个人信息。 如您未满 14 岁，请在监护人陪同下使用，并避免使用表单评论功能。


五、使用限制

5.1 短链接限制

为避免纠纷，未经齐喜允许，禁止使用短链接生成平台生成跳转至我们网站的短链（如，45d.cn）

5.2 外部链接声明

您在本网站外（包括其他网站、小程序、下载的文件等）的一切行为与本网站无关。 齐喜不对外部链接的内容做任何保证。

5.3 设备适配

提示：本网站 UI 页面主要适配移动端，桌面端访问可能存在显示异常。

这不是网络游戏，但请合理安排使用时间。 您可能不会得到访问某些子目录网页的权限，或可能遇到访问限制。


六、用户承诺

点击"我同意"即代表您承诺：

  • 已知晓本站的娱乐性质，不将其作为专业指导依据
  • 不会模仿虚拟产品的危险行为，不在现实中尝试复刻
  • 理解外部链接内容由第三方负责，与本站无关
  • 遵守中华人民共和国法律及您所在地区的适用法律
  • 如使用评论功能，内容遵守社区规范，不发布违法信息
  • 知晓部分内容由 AI 生成，仅供娱乐，不具有代表性

协议更新

如本协议有改动，通常会再次弹出本窗口，您需要再次阅读并选择是否继续使用。 继续使用视为接受新条款。
